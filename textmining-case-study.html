Dataset: 100 Wikipedia article summaries scraped programmatically. Text corpus of ~80,000 tokens processed through full NLP pipeline (tokenization, NER, sentiment, co-occurrence network).

Project Overview

This end-to-end NLP project scrapes a Wikipedia article on 20th century key events, performs text mining and linguistic analysis, then uses Named Entity Recognition (NER) to extract country co-occurrences and build an interactive network graph showing which nations appeared together most frequently in historical events.

The Pipeline

The project covers the full data science pipeline from raw web data to interactive visualization — scraping, cleaning, linguistic analysis, entity extraction, relationship modeling, and network graph construction.

Phase 1 — Web Scraping with Selenium

Used Selenium with ChromeDriver to scrape the full text of the Wikipedia "20th century" article programmatically.

from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://en.wikipedia.org/wiki/20th_century")
text = driver.find_element(By.ID, "mw-content-text").text
driver.quit()

with open("20th_century_key_events.txt", "w") as f:
    f.write(text)

Phase 2 — Text Mining with NLTK & TextBlob

  • Word frequency analysis — before and after stopword removal revealed "war", "world", and "soviet" dominate
  • Part-of-speech (POS) tagging using TextBlob
  • Top 15 nouns extraction to surface the most historically significant themes

Phase 3 — Named Entity Recognition with spaCy

Used spaCy's en_core_web_sm model to extract geopolitical entities (GPE tags). Entity aliases were normalized — "UK", "U.K.", and "Britain" all mapped to "United Kingdom".

import spacy

nlp = spacy.load("en_core_web_sm")
doc = nlp(clean_text)

sentence_entities = []
for sent in doc.sents:
    entities = [ent.text for ent in sent.ents if ent.label_ == "GPE"]
    if len(entities) > 1:
        sentence_entities.append(entities)

Key Findings

  • After stopword removal, most frequent words: "war", "world", "soviet", "united", "states"
  • US, Soviet Union, Germany, and UK formed the most densely connected cluster
  • Country co-occurrence patterns closely mirror known geopolitical relationships
  • Alias normalization was critical — without it, "US", "U.S.", and "United States" appear as 3 separate nodes
