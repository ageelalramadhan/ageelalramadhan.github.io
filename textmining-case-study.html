20th Century Text Mining & NLP

Python · Selenium · NLTK · TextBlob · spaCy · NetworkX
Instacart Citi Bike Rockbuster Influenza Happiness Text Mining
Tools & Libraries
Python · Selenium · NLTK · TextBlob · spaCy · NetworkX · NER

Dataset: 100 Wikipedia article summaries, ~80,000 tokens processed through full NLP pipeline.

Project Overview

End-to-end NLP pipeline: web scraping historical Wikipedia text, named entity recognition, and network graph visualization of geopolitical co-occurrence patterns across 20th century events.

Phase 1 — Web Scraping with Selenium

from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://en.wikipedia.org/wiki/20th_century")
text = driver.find_element(By.ID, "mw-content-text").text
driver.quit()

with open("20th_century_key_events.txt", "w") as f:
    f.write(text)

Key Findings

  • Top words after stopword removal: "war", "world", "soviet", "united", "states"
  • US, Soviet Union, Germany, and UK formed the most densely connected cluster
  • Alias normalization was critical — "US", "U.S.", "United States" must map to one node
